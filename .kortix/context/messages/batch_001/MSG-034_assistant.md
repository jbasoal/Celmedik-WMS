1. **Crear cuenta en Firebase** (gratis) - https://console.firebase.google.com
2. **Copiar tu configuracion** en el archivo `wms-nube.html`
3. **Crear cuenta en GitHub** (gratis) - https://github.com
4. **Subir archivos** a un repositorio
5. **Activar GitHub Pages** para publicar automaticamente

**Costos:** Todo es completamente **GRATIS** (Firebase tiene limite gratuito de 100,000 usuarios/mes)
